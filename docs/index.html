Exprify - Math Expression + Simplify

Exprify is a JavaScript expression parser and evaluator. It is made for math applications, scientific computing, and complex workflows in the browser and Node.js. It supports basic arithmetic, variables, and defined functions. Exprify also covers advanced mathematical areas like unit conversion, matrix operations, complex number arithmetic, and symbolic manipulation.

Installation

npm

npm install exprify

CDN

<script src="https://unpkg.com/exprify"></script>

API Reference

new Exprify()

Creates a new expression engine instance with its own variable and function stores.

import Exprify from 'exprify';
const expr = new Exprify();

.evaluate(expr, scope?)

Parses and evaluates an expression string. Returns the result (number, string, array, object, etc.).

expr.evaluate('2 + 2 * 3');         // 8
expr.evaluate('x ^ 2', { x: 5 }); // scope override

.compile(expr)

Compiles an expression into a reusable function. Cached internally.

const fn = expr.compile('a * b + c');
fn({ a: 2, b: 3, c: 1 }); // 7

.parse(expr)

Returns { tokens, ast } — the token array and AST for an expression.

.setVariable(name, value) / .getVariable(name)

Set or retrieve persistent variables.

expr.setVariable('x', 42);
expr.getVariable('x'); // 42

.addFunction(name, fn)

Register a custom function for use in expressions.

expr.addFunction('double', x => x * 2);
expr.evaluate('double(21)'); // 42

.exportState() / .importState(state)

Serialize and restore the engine state (variables, functions, units).

const state = expr.exportState();
expr.importState(state);

.chain()

Returns a fluent Chain object for chained evaluation. Each step stores its result as ans for the next step.

const c = expr.chain();
c.evaluate('2 + 2')           // ans = 4
 .evaluate('ans * 10')       // ans = 40
 .evaluate('ans / 2');       // ans = 20
c.done();                        // 20

Features

Arithmetic

Standard operators: + - * / ^ %, parentheses, implicit multiplication, unary plus/minus.

expr.evaluate('2 + 3 * 4 ^ 2'); // 50

Variables & Assignments

expr.evaluate('x = 10');
expr.evaluate('y = x * 2');    // 20

Inline Functions

expr.evaluate('f(x) = x ^ 2 + 1');
expr.evaluate('f(5)');            // 26

Unit Conversion

expr.evaluate('2 inch to cm');   // 5.08 cm
expr.evaluate('5 cm + 2 inch');

Matrices

expr.evaluate('m = [1, 2; 3, 4]');
expr.evaluate('det(m)');            // -2
expr.evaluate('inverse(m)');

Fractions

expr.evaluate('fraction(1, 3) + fraction(1, 6)'); // "1/2"

Arbitrary Precision (BigNumber)

expr.evaluate('bignumber("0.1") + bignumber("0.2")'); // "0.3"
expr.evaluate('bignumber("1.2e500")');                // "1.2e+500"

Symbolic Algebra

expr.evaluate('simplify("x^2 + 2x + 1")');
expr.evaluate('expand("(x+1)^2")');
expr.evaluate('factor("x^2 - 5x + 6")');

Calculus

expr.evaluate('derivative("x^3", "x")');  // "3x^2"
expr.evaluate('integral("x^2", 0, 1)');     // 0.3333

Statistics

expr.evaluate('mean(1, 2, 3, 4, 5)');   // 3
expr.evaluate('std(1, 2, 3, 4, 5)');    // 1.5811

Expression Chaining

const c = expr.chain();
c.evaluate('2 + 2')
 .evaluate('ans * 10')
 .evaluate('ans + bignumber("0.5")');
c.done(); // "40.5"

Degree Trig

expr.evaluate('sind(90)');  // 1
expr.evaluate('cosd(0)');   // 1
expr.evaluate('tand(45)');  // 1

State Serialization

const state = expr.exportState();
const expr2 = new Exprify();
expr2.importState(state);
